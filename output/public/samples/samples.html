Sample Apps on Cloud Foundry

Tags: samplesexamples

Last Updated: 2012-10-08

Node.js

App Source Description Service(s) Used
Octofoundry https://github.com/videlalvaro/cloudfoundry-octopress Deploy your octopress blogs to Cloud Foundry.
Node Activities Boilerplate https://github.com/cloudfoundry-samples/node-activities-boilerplate A simple NodeJS activity stream engine built with Express, Connect, Socket.io, Redis and MongoDB. MongoDB, Redis
Activity Streams Mongoose https://github.com/cloudfoundry-samples/activity-streams-mongoose Activity Streams Real-Time Data Store backed by MongoDB (via Mongoose) and Redis. MongoDB, Redis
Activity Streams Mongoose https://github.com/cloudfoundry-samples/subway Subway is a web-based IRC client with a multi-user backend and a JavaScript-heavy UI. MongoDB
Twitter Rabbit Socks sample https://github.com/cloudfoundry-samples/twitter-rabbit-socks-sample This contains a node.js application that consumes tweets from a Rabbit queue and pushes them to the browser using Sock.js. RabbitMQ
Solr Node sample https://github.com/cloudfoundry-samples/solr-node-sample This is a sample easy to use search web application that sits on top of Apache Solr.

Java

App Source Description Service(s) Used
SpringMVC Hibernate Template https://github.com/cloudfoundry-samples/springmvc-hibernate-template A simple application that demonstrates how to use Spring 3.1 on Cloud Foundry with profiles, HTML5, caching and Hibernate 4.0, and more.
Stock Workers https://github.com/cloudfoundry-samples/stock-workers This repository hosts the demos for the SpringSource blog on workers in Cloud Foundry. The demos - Spring Batch, Spring Integration and Spring Core, consume and work with stock feed data. MySQL
Play Sample app https://github.com/cloudfoundry-samples/todolist-play-java-mongodb This is a Play Sample App using Java and MongoDB. MongoDB
Twitter Rabbit Socks sample https://github.com/cloudfoundry-samples/twitter-rabbit-socks-sample This contains a standalone Spring Integration application that periodically polls twitter for tweets containing the word "cloud" and places them in a Rabbit queue. RabbitMQ
Spring Travel https://github.com/cloudfoundry-samples/spring-travel Spring Travel Reference Application - demonstrates Spring Framework 3 with Web Flow 2.1

Scala

App Source Description Service(s) Used
Spray Can Server https://github.com/cloudfoundry-samples/spray-can-server The spray-can simple-http-server example from Spray. Spray is a suite of scala libraries for building and consuming RESTful web services on top of Akka. This example is deployed to Cloud Foundry as a standalone application.
Unfiltered sample https://github.com/cloudfoundry-samples/cf-unfiltered-sample An Unfiltered sample generated from the giter8 template. Unfiltered is a toolkit for servicing HTTP requests in Scala. This example uses embedded Jetty and is deployed to Cloud Foundry as a standalone application.
Bitshow sample https://github.com/cloudfoundry-samples/bitshow Image processing server. This is a clone of the nyscala-bitshow project, modified to work on Cloud Foundry. MongoDB
Scalatra Mongo sample https://github.com/cloudfoundry-samples/scalatra-mongo-sample Simple Scalatra app that stores and retrieves messages from Mongo. MongoDB

Ruby

App Source Description Service(s) Used
Contributing Code https://github.com/cloudfoundry-samples/contributing-code-app This app was primarily built to organize a coding event. It is a Rails 3.2.6 app using MySQL for storing data, Mongo GridFS to store images, Sendgrid API for email delivery and Redis for Resque operations. MySQL, Redis
Backbone cars https://github.com/cloudfoundry-samples/backbone-cars A demo application for CabForward's Backbone Class. MySQL
Rails Elastic Search https://github.com/cloudfoundry-samples/rails-elastic-search A Rails app that uses Tire to interact with Elastic Search deployed on Cloud Foundry. MySQL
Rails Pre-launch Signup https://github.com/cloudfoundry-samples/rails-prelaunch-signup An example Rails 3.2 app for a web startup prelaunch site. MySQL
Resque sample https://github.com/cloudfoundry-samples/resque-sample A resque demo sample with Gemfile and Cloud Foundry manifest files. Highlights CF Ruby auto-reconfiguration by automatically connecting the server and workers to a CF Redis service. Redis
Summer jobs https://github.com/cloudfoundry-samples/summer-jobs Showcases how to use Summer Jobs API from the US DOL and how to use the Job Schema with Microdata. Redis
Friends https://github.com/cloudfoundry-samples/friends Facebook App which allows you to hook your (younger) friends up with Summer Jobs. Redis
Sinatra Twitter https://github.com/cloudfoundry-samples/sinatra-cf-twitter A Sinatra/Redis app - does one Twitter user follow another? Redis
JRuby Rails Bookshelf https://github.com/cloudfoundry-samples/jruby-rails-bookshelf Rails JRuby sample for Cloud Foundry.
Enki https://github.com/cloudfoundry-samples/enki A Ruby on Rails blogging app for the fashionable developer. MySQL
Sinatra Cloud Foundry Basic Website https://github.com/cloudfoundry-samples/sinatra-cloudfoundry-basic-website Super simple hello world sinatra app. Haml used for views. Also added some Open Graph Protocol for fun.
Salesforce demo https://github.com/cloudfoundry-samples/salesforce-demo Sample REST Client for Salesforce which uses OAuth2 gem. Redis
Box Sample app https://github.com/cloudfoundry-samples/box-sample-ruby-app A sample app built with the Box Ruby SDK.

App Source Description Service(s) Used
Grails Twitter https://github.com/grails-samples/grailstwitter A Grails Twitter sample application. MongoDB
